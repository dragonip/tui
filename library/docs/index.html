Loads CSS files in the document if they are not already loaded, functions changed after 2beta and now uses XHR to get the CSS, thus you need to host it in the same domain as the main application. Alternative transport is allowed (loadExternalCSS) that uses a trick with image load even to notife when the css is done loading

{Array} List of urls with CSS to load
{Function} Callback to execute when all CSS is loaded
loadCSSFromText

Loads css rules from string, used in modules that get the scc via require/define text plugin

{String} The CSS rules as text
{String} [optional] Id to set in the style tag, if one is provided we check to see if we already have one and if we do we skip this one
loadExternalCSS

Used to load CSS that is not from the same domain, it will use the link attribute and a 'hack' to call the callback function once the CSS is loaded

{String} The url of the css file to load in the document
{Function} The callback function to call when the file is loaded in the document
